# CubeHub-Launcher
一个现代化的Minecraft启动器，拥有现代化的UI界面和界面动效，基于Tauri项目开发，UI界面有着较高的流畅性

当前代码属于Beta版本，暂未完成所有功能开发
为了确保运行正常，请手动解压src-tauri.7z到启动器根目录，确保目录结构如下

CubeHub Launcher.
├─src
│  ├─assets
│  │  └─IMG
│  │      ├─Fonts
│  │      ├─Icons
│  │      └─Skins
│  ├─components
│  ├─composables
│  ├─router
│  ├─services
│  ├─stores
│  ├─types
│  ├─utils
│  └─views
│      ├─download
│      ├─settings
│      └─Start
└─src-tauri
    ├─capabilities
    ├─icons
    └─src
